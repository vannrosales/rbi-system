🌟 System Overview: Barangay 3 RBI Manager (CY 2026)
This project is a high-performance Registry of Barangay Inhabitants (RBI) management system tailored for the specific administrative needs of Barangay 3. It moves away from traditional, slow database entry by using a "Family-First" Vibe Coding approach—allowing staff to register entire households in a single, fluid motion.

🛠️ Key Built Features
1. Smart Family Grouping (One-Form Entry)
Shared Context: Enter the Region, Province, Municipality, and Street once at the top.

Dynamic Stacking: Use the "+ Add Member Card" to instantly stack family members (Parents/Children) under the same address.

Intelligent Defaults: Automatically carries over the Parent’s Last Name and suggests "Child" as the relationship for new entries to minimize typing.

2. Official CY 2026 PDF Engine
Selective Export: A "Check-to-Print" system where you can select specific individuals or entire families from the ledger.

Government Format: Exports directly into the Barangay Registered Inhabitants' CY 2026 landscape format, complete with official headers and categorized columns.

3. Compliance-Ready Categorization
Strict Indications: Integrated dropdowns for official categories: Labor/Employed, PWD, OFW, Solo Parent, OSY, OSC, and IP.

Real-time Bio-Sync: Automatically calculates age from the Date of Birth to ensure 100% accuracy in Senior Citizen and Voter population counts.

4. High-Contrast "Staff-Friendly" UI
Large Typography: Optimized for visibility with bold headings and clear labels.

Live Search: A lightning-fast, client-side search bar that filters the ledger by name, street, or ID without page refreshes.

Action-Ready Ledger: Hover-state buttons for Edit/Delete to keep the interface clean but powerful.


🔧 Technical Core
Backend: Laravel 11 (PHP 8.3)

Frontend: React 19 + Inertia.js 

Styling: Tailwind CSS 4 (Clean Light Mode)

PDF Logic: jsPDF + AutoTable (Landscape Optimized)
